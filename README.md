# SCUTTLE™ Robot
This repository contains software (embedded Linux & Python) and hardware design files for the SCUTTLE Project.

### Sensing, Connected, Utility Transport Taxi for Level Environments
* payload capable (40kg)
* maker-friendly
* open source
* low-cost
